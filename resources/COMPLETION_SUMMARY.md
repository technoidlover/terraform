## Getting Started

### Quick Start (5 minutes)

```bash
# 1. Read the main README
type README.md

# 2. Follow Quick Start Guide
type resources\QUICKSTART.md

# 3. Navigate to Day 1
cd day-01-terraform-basics-and-setup

# 4. Read the day's guide
type README.md
```

### Start Learning (30 minutes)

```bash
# 1. Install Terraform if needed
terraform -v

# 2. Configure AWS/cloud credentials
aws configure

# 3. Test configuration
cd day-01-terraform-basics-and-setup
terraform init
terraform validate
```

### Complete Day 1 (3-4 hours)


- "I need to memorize everything" - No, understanding is key
- "I must deploy everything to production" - No, use free tier/dev environment
- "Terraform is just for AWS" - No, works with 100+ providers
- "One way to do things" - No, multiple valid patterns exist
