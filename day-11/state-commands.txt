# State Management Practice Commands
# Cac lenh Thuc hanh Quan ly State

# After applying the configuration, practice these commands:
# Sau khi ap dung cau hinh, thuc hanh cac lenh nay:

# 1. List all resources in state
# 1. Liet ke tat ca resources trong state
# terraform state list

# 2. Show details of a specific resource
# 2. Hien thi chi tiet cua mot resource cu the
# terraform state show local_file.file_a

# 3. Move a resource to new name
# 3. Di chuyen resource sang ten moi
# terraform state mv local_file.file_a local_file.renamed_file_a

# 4. Remove a resource from state (doesn't delete the actual file)
# 4. Xoa resource khoi state (khong xoa file thuc su)
# terraform state rm local_file.file_b

# 5. Pull current state and save to file
# 5. Keo state hien tai va luu vao file
# terraform state pull > state-backup.json

# 6. View state in human-readable format
# 6. Xem state o dinh dang de doc
# terraform show

# 7. Refresh state to match real infrastructure
# 7. Lam moi state de khop voi ha tang thuc
# terraform refresh

# Important Notes:
# Luu y Quan trong:
# - Always backup state before manipulation
# - Luon sao luu state truoc khi thao tac
# - Use state commands carefully
# - Su dung cac lenh state mot cach can than
# - Verify changes with 'terraform plan' after
# - Xac minh thay doi voi 'terraform plan' sau do
